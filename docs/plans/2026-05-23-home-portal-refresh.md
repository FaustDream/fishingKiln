# 2026-05-23 首页门户化重构计划

## 背景

当前首页仍残留专题首页的组织方式，一级阅读路径过度依赖“专题/研究索引”，与“专题只作为二级页面进入”的要求不一致。页面已有较多结构化数据和官方来源，但首页渲染没有把这些数据组织成正式门户页。

## 目标

把首页重构为正式的网站首页：

- 以青花历史、景德镇、馆藏来源和器物栏目作为一级入口
- 使用列表数据和卡片数据混排，形成高校/国际组织官网式首页节奏
- 提供一套首页通用内嵌详情面板，承接来源条目、重点对象、时间线与术语
- 移除首页对专题页的一级分发，只保留分类页入口
- 保留轻量动效，并完成响应式回归

## 非目标

- 不重写分类页主体结构
- 不引入新依赖
- 不改动专题详情页的数据模型

## BDD 验收场景

### 场景 1：首页作为总站入口
Given 用户进入首页  
When 页面完成渲染  
Then 用户先看到总站级标题、栏目总览、资料台账和器物入口  
And 不再看到首页一级“专题分类”分发

### 场景 2：首页内嵌详情联动
Given 用户位于首页  
When 点击资料台账、重点对象、时间线或术语卡  
Then 右侧或下方的通用详情面板更新为对应内容  
And 不需要跳出当前页面

### 场景 3：首页数据来源可追溯
Given 用户查看首页资料节点  
When 展开任一详情  
Then 可看到来源机构、关键事实和原始来源链接

### 场景 4：分类页继续承担二级深读
Given 用户从首页进入任一器物栏目  
When 打开分类页  
Then 仍能从分类页继续进入专题页  
And 首页本身不承担专题页一级导航

### 场景 5：多端布局稳定
Given 用户在手机、平板和桌面访问首页  
When 页面加载完成  
Then 列表、卡片、详情面板和栏目入口都不溢出、不重叠  
And 动效不会导致内容长时间不可见
